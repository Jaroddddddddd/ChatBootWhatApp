# ChatBootWhatApp

**Tema:** Gimnasio

**Pasos de trabajo complementario**
1.-Levantar el docker
2.-Ubicar un archivo de dockerfile en el directorio raiz.
3.-Configurar el archivo docker, donde se usaran capas para que esto funcione.
4.-Construimos la imagen del docker con el comando:  docker build . -t complementario-6
5.- Se comprueba que se haya creado la imagen en el docker desktop.
6.- Ejecutamos el comando docker run para que se convierta en un contenedor con el siguiente comando: docker run --name contenedorcomplementario6b -d complementario-6
7.- Nos muestra el resultado con un ID y saldra en verde que se encuentra ejecutando.
8.- Eliminamos lo creado para que no consuma espacio.
